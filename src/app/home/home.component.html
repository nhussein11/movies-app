Welcome!

This project was created with Angular and PrimeNG. The main purpose of it is to show how to use PrimeNG components in an Angular project, how to use PrimeNG themes and Prime Flex css framework, apply Redux pattern with NGRX library and styling some components with SASS.
Currently,this project get the data from The Movie Database API, which is a free service for movies and TV shows. You can find more information about it here: https://www.themoviedb.org/documentation/api
Of course, if you want to use this project as a base for your own project, feel free to do it.
